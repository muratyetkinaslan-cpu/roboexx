DONANIM GALERİSİ İKONLARI — PNG yerleştirme
============================================

Bu klasöre sensör/aktüatör PNG'lerini şu isimlerle koy:

  led.png         → LED
  rgb.png         → RGB LED
  buzzer.png      → Buzzer
  servo.png       → Servo
  dcmotor.png     → DC Motor
  button.png      → Buton
  ultrasonic.png  → Mesafe Sensörü
  oled.png        → OLED Ekran

Öneriler:
  - Kare format (örn. 64x64 veya 128x128 px)
  - Şeffaf arka plan (PNG transparency)
  - Dosya boyutu küçük tut (< 50 KB ideal)

PNG koymazsan renkli çizim ikonlar gösterilir — uygulama yine çalışır.
PNG'yi koyduktan sonra tarayıcıyı yenile, otomatik görünür.
